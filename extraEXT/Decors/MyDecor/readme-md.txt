# readme-md.txt

This decor is extracted from addy-fe's Diary theme.
Original download: https://drive.google.com/file/d/1gLhWQBP7cr61az6qWwZeLNXeF6KTALm5/view

To install or edit .fvwm/ExtraEXT.sys

Add the below line below "Decorations".
Read $[EXTRA_DIR]/Decors/MyDecor/myDec.sys #501

Install dependency packages.

Add the below lines below "Styles".
Read $[EXTRA_DIR]/Functions/Thumbnails/thumbnail.sys #103
Read $[EXTRA_DIR]/Styles/MyBorderStyle/myBorderStyle.sys #201

## Any addon or modification in .fvwm/UserEXT.sys

## Fvwm Kise

Copy .fvwm/extraEXT/Decors/FancyDecor to .fvwm/user.

IMPORTANT:
Edit/untag where there is an Fvwm Kise label.
