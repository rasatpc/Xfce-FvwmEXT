# readme-dsw.txt

To install or edit .fvwm/CoreEXT.sys

## Add the below line below "Load 7.Modules".
Read $[CORE_DIR]/MODULES/DeskSwitch/deskSwitch.sys

## Fvwm Kise

Copy .fvwm/coreEXT/MODULES/DeskSwitch to .fvwm/user. Add this line in .fvwm/userExt.sys in MODULES section.
Read $[USER_DIR]/DeskSwitch/deskSwitch.sys

IMPORTANT:
Edit/untag where there is an Fvwm Kise label.
